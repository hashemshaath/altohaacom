

# Altohaa — Culinary Community Platform (Phase 1)

**Brand:** Altohaa — with the uploaded logo, warm & elegant design (peach, cream, brown tones), bilingual (English + Arabic with RTL support)

---

## Phase 1: Foundation — Landing Page, Authentication & User Profiles

### 1. Landing Page
- **Hero section** with Altohaa logo, tagline about the culinary community, and call-to-action buttons (Join / Sign In)
- **Features overview** — cards highlighting key platform pillars: Community, Competitions, Masterclasses, Culinary Shop
- **Language switcher** (EN/AR) in the header with full RTL support for Arabic
- **Footer** with links, social media, and Altohaa branding
- Warm, elegant design palette matching the logo (peach #F5DEB3, cream, dark brown accents)

### 2. Authentication System (Backend Required)
- **Sign Up page** with role selection: Chef, Judge, Student, Organizer, Volunteer, Sponsor, Assistant, Supervisor
- **Sign In page** with email/password
- **Profile completion flow** after first sign-up — collect name, photo, bio, specialization, experience level (beginner/amateur/professional), location, social media links
- Proper session management and protected routes

### 3. User Profile Pages
- **Public profile view** — photo, name, role badge, bio, specialization, social media links, experience level
- **Edit profile** — form to update all personal details
- **Role-based badge/tag** displayed on profile (e.g., "Professional Chef", "Judge", "Student")
- Profile page designed as a chef's professional card

### 4. Basic Navigation & Layout
- **Responsive sidebar/header** with navigation links: Home, My Profile, Competitions (placeholder), Community (placeholder), Shop (placeholder)
- **Language toggle** persisted across pages
- **Mobile-responsive** design throughout
- Consistent warm & elegant design system applied globally

### 5. Bilingual (EN/AR) Support
- i18n setup with language context provider
- All UI text available in English and Arabic
- Automatic RTL layout when Arabic is selected
- Language preference saved per user
